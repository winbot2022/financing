# 資金繰り表 作成アプリ

## 概要
信金・銀行で使用される「資金繰り表」を、スマホでも素早く入力し、Excelで出力するためのアプリです。

## 特徴
- 開始月を選択
- 6か月固定で入力
- 区分は「収入 / 支出 / 財務」
- 分類・科目はプルダウン選択
- 金額はテンキー方式で入力
- 前月繰越を起点に自動計算
- 資金繰り表をExcelで出力
- 出力したExcelを再読込して続きから再開可能
- 残高推移グラフを表示

## 主な用途
- 経営者との資金繰り確認
- 金融機関提出用の資金繰り表作成
- 面談時のその場入力・その場出力

## 起動方法

streamlit run app.py
